Topics

This exercise is challenging, but doable with only lists (of lists)! Lists of lists are nearly the same as lists, just a bit trickier. Remember that to get the first element in a list called my_list = [5, 10, 15], you index it with a variable like so:

When working with lists of lists, it is the same! Say you have a list matrix = [[1, 2], [3, 4]]. Then, take a look at this:

The cool part is that you can use double-indexing to read the elements from our list of lists matrix! By indexing matrix[0][1] we are saying give me the 1st element of the 0th element of matrix, which in this case is 2.

這個活動是屬於Tic Tac Toe活動系列4的第二部分，其他系列的活動分別是:第一部分、第三部分和第四部份。

你可能已經猜到，我們正在努力建構井字遊戲。但是這很明顯的已經超過半個小時的編碼，因此我們要分批進行。

今天，我們只須關注某人贏得比賽，而不需要知道他怎麼贏得比賽的。

如果Tic Tac Toe的遊戲在列表裡又被表示為列表，像是:

game = [[1, 2, 0],
	[2, 1, 0],
	[2, 1, 1]]

其中a0為一個空的方格，a1表示玩家1將其令牌放置的位置，a2表示玩家2將其令牌放置的位置。
你本周的任務為讓玩家3把他的令牌放到a3的位置，如果有玩家贏了，請告訴我。Tic Tac Toe的規則是3個連成一條線，不管是一列、一行或一個對角線，就算贏了，假如每組只有一個玩家，就不需要擔心會有兩個玩家同時贏得比賽。

以下有一些範例:

winner_is_2 = [[2, 2, 0],
	[2, 1, 0],
	[2, 1, 1]]

winner_is_1 = [[1, 2, 0],
	[2, 1, 0],
	[2, 1, 1]]

winner_is_also_1 = [[0, 1, 0],
	[2, 1, 0],
	[2, 1, 1]]

no_winner = [[1, 2, 0],
	[2, 1, 0],
	[2, 1, 2]]

also_no_winner = [[1, 2, 0],
	[2, 1, 0],
	[2, 1, 0]]

主旨

這個活動有一些挑戰性，譬如只能使用表單中的列表，表單中的列表和其它列表很相近，只是比較棘手。記得，想要取得名為 my_list = [5, 10, 15]的關鍵字，你必須造著底下的步驟:

>>> print(my_list[0])
	5
	>>> print(my_list[-1])
	15
	>>> print(len(my_list))
	3
使用表單中的列表時，步驟是一樣的。打出你想要的東西，然後來看看底下的範例:
>>> print(matrix[0])
	[1, 2]
	>>> print(matrix[-1])
	[3, 4]
	>>> first_row = matrix[0]
	>>> print(first_row[0])
	1
	>>> print(matrix[0][0])
	1
	>>> print(matrix[1][1])
	4
最酷的部分為你可以使用雙索引來讀取表單中的列表元素!透過元素[0][1]的索引，我們可以跟它說請給我列表1中，位於a3的0。
